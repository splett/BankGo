Esse projeto foi realizado em Java1.8 e utiliza Maven para gerenciamento de dependências. 
Utiliza a biblioteca externa: 
 - Jersey, um frameworks para desenvolver aplicações REST em Java

Para realizar rodar o projeto no Eclipse siga os seguintes passos: 
 - Se ainda não possuir, baixe o Eclipse Oxygen.3a Release (4.7.3a) em https://www.eclipse.org
 - Abra o Eclise 
 - Baixe o arquivo em uma pasta local e descompacte na pasta do workspace
 - Na aba Package Explorer, clique com o botão direito e selecione "Import" 
 - Selecione a opção "Existing Projects into Workspace"
 - Selecione a pasta do projeto e finalize
 - Atualize as dependências do Maven caso não tenha baixado automaticamente
 - Para rodar a aplicação selecione o projeto, botão direito, selecione "Run as -> Run on Server" e por fim selecione o servidor Tomcat

A documentação está disponível na pasta doc. Para visualizar clique index.hmtl e abra com o navegador.
O WAR gerado para esse projeto está disponível na pasta war.
